# Valoshot Model IDs

## ベースアイテム
全ての武器モデルは **`minecraft:iron_hoe`** (鉄のクワ) をベースにしています。

## メモ
- これらのIDは `config.yml` 内で定義されており、変更可能。
- リソースパックの `assets/minecraft/models/item/iron_hoe.json` 内の `predicate` 設定と一致させる必要がある。
